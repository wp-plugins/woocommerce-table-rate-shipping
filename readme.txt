== Description ==

This plugin allows provide you the option to set the shipping rate based up on the cart price. You can set the cart price range and set the shipping cost corresponding to each range. If any of the range is not availabe then shipping cost will be free for that plugin. Admin can also change the title for the same.



== Installation ==

1. Upload `woocommerce-table-rate` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Plugin will add ths Table rate shipping options in the Woocommerce shipping tab.
